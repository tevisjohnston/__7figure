# Project: 7 Figure Affiliate

## 1. Project Overview

**Goal** Create a high-converting affiliate marketing WordPress website called "7 Figure Affiliate"
**Owner/Brand** Michael Cheney
**Niche** Make Money Online, affiliate marketing  digital products hosted on `warriorplus.com`
**Core Function** Serve as a dedicated website for promoting a set of landing pages for 10 products created by Michael Cheney. Each landing page is for driving traffic to the product's affiliate link via Call to Action (CTA)
**Tone** Persuasive, professional, profit-focused, and highly credible, leveraging Michael Cheney's established reputation

## 2. Technical Stack & Environment

**WordPress** PHP, HTML, Javascript, TailwindCSS - Custom Theme development (Theme Name: "7 Figure Affiliate")
**Development** Focus on standard classic theme WordPress development practices
**Database** `http://localhost:8888/phpMyAdmin5`
**Browser** `http://localhost:8888/7figure.affiliatemarketconnect.com`

## 3. Front-End Structure

**Home Page** front-page.php - Header/Navigation, Hero Section with lead magnet "Step-by-Step Blueprint to $5k/Month", About Section (with Michael Cheney bio image), 3-Card Grid for 3 Featured Products, Testimonials Section, Footer
**Blog Home** home.php, sidebar.php - Standard WordPress blog layout with Hero Section "7 Figure Insider", Post Carousal with most recent posts, and sidebar with email sign up form "7 Figure Insider Secrets"
**Blog Post** single.php - Template for each blog post
**Products Page** page-products.php - Page listing all 10 products inside individual product cards with titles, short descriptions, a button that links to the single-product pages, and a button that links to the affiliate link
**About Page** page-about.php - Expanded bio on Michael Cheney, mission statement, and authority building content
**Contact Page** page-contact.php - Simple form and contact information
**Testimonials Page** page-testimonials.php - Archive/list of all testimonials (can be managed via a CPT or dedicated page structure)
